The Project Reveal!

This lecture we learned what the project is about: CRUD Apps! More specifically, the overarching project is to create a journal/diary of what you did as a software engineer. Kinda like a timeline of events.

  • This seems as an interesting choice, and using a seemingly simple project to teach the idea of a CRUD app initially doesn't sound as the right way to do it. CRUD stands for:
    1. Create
    2. Read
    3. Update
    4. Delete
  • Because almost all apps are CRUD apps, this means that we already know what a CRUD app essentialy should do, and we to figure out a way to make it work for us as a team as well as make it stand out.
